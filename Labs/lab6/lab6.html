A favicon is an icon that shows up next to a website's title, which is usually seen up on the tab within a web browser. For example, the Apple website has their signature apple logo as their favicon. Favicons can be created through binary coding and the colors of the pixels are coded through RGB in bits of 3 or more, if you want more complementary colors, like how red is 100, blue is 001, and green is 010. The html tag for favicon between the opening and closing head tag and typed as, open angle bracket link rel equals, quotation marks, shortcut icon, end quotes, then include a space, href equals, quotations, name of the png or jpg file type, end quotes. In the same tag, it continues with type equals, "image/x-icon", then space, adding the width usually needing to be 32 pixels to appear as the symbol for your webpage.

I made a favicon for this website by first creating a drawing of my logo on graph paper with a 32 by 32 box. Then I went to code.org to apply my written idea online within black and white pixels. It went by 1 bit per color, with 0 being black and white being 1, finishing my second design. Then the final draft of my favicon was using 3 bits for multiple colors within binary coding, attaining an end result of a 32px sized logo for my webpage. The code was, angle bracket, link rel="shortcut icon" href="download.png" type="image/x-icon" width="32px", close angle bracket.
