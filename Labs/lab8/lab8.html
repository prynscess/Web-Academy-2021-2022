External CSS, which stands for Cascading Style Sheets, is used for design and presentation purposes when coding websites. It helps the web development process because the navigation bar provides convience and order for switching from lab to lab. The header's style, background color, font size, boxes and more can be coded through CSS, so the way things are display can be relied on this function. Although the factors mentioned can also be made through regular HTML arrangment, there's more ease for CSS since it is a separate file from HTML, it doesn't get messy within the original content you want to share.
